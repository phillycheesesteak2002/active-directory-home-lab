# Active Directory Home Lab

## Overview

This project demonstrates the installation and configuration of Active Directory Domain Services using Windows Server 2022 and Windows 10 client in VirtualBox.

This lab simulates real-world Help Desk and System Administrator tasks.

---

## Technologies Used

• Windows Server 2022  
• Windows 10  
• Active Directory Domain Services  
• Oracle VirtualBox  
• Command Prompt  

---

## Lab Objectives

• Install and configure Active Directory  
• Create domain users  
• Join Windows 10 client to domain  
• Reset user passwords  
• Verify domain login authentication
